Image Threshold Optimizer
==========================
## Features
- Automatic threshold optimization for image binarization
- Standard deviation and histogram analysis
## Project Structure
- main.py: Main script for threshold optimization
- image_utils.py: Utilities for image processing
- requirements.txt: Dependencies for the project
## Installation
- pip install -r requirements.txt
## Usage
- python main.py <image_path>
## Roadmap
- Integrate with AIERA CAD for real-time threshold optimization
